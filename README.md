# react-native-wavy-text

Enrichment for Text component with a new underline style

## Installation

```sh
npm install react-native-wavy-text
```

## Usage

```js
import { WavyTextView } from "react-native-wavy-text";

// ...

<WavyTextView color="tomato" />
```
